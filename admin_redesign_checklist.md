# admin_page.dart 재설계 체크리스트

기사페이지(driver_page.dart) 스타일·구조를 그대로 참고. 전체배경 → 메인배경 패널 → 탭/카드/버튼 구조.

## 공통 규칙
- [ ] 색·숫자·여백·테두리 전부 상단 토큰으로 분리 (글씨 크기 / 숫자 크기 따로)
- [ ] 카드는 항상 오른쪽+아래 그림자 (`_cardShadow`)
- [ ] 이모지 금지, 아이콘만 사용
- [ ] 팔레트: 네이비+민트 (driver와 동일)

## 세션 분할 (실제 진행)
- [x] **S1 토대**: 팔레트 네이비+민트 이식
- [x] **S2 대시보드**: 패널 + 인사 + 메뉴 그룹 카드(기존 페이지 연결, badge 유지)
- [x] **S3 차트 카드**: 누적 지급액(일/주/월) + 링 게이지 + 목표치 편집
- [x] **S4 출금 랭킹 TOP3**: 주간/월간 토글 + 더보기(전체회원 페이지)
- [x] **S5 서브페이지 패널**: 공통 _adminPanelScaffold 전 페이지 적용
- [x] **S6 출금신청 카드**: 외형/헤더/계좌·최종금액 토큰화 + 그림자
- [x] **S7 라이더관리 / 리스비**: 카드 그림자 + 외형 토큰
- [x] **S8 공제설정**: 카드 그림자
- [x] **S9 공지사항/1:1상담**: 공지 카드들 그림자 (상담은 패널 구조)

## 결정됨
- 차트: admin_settlement_logs '지급완료' amount 누적
- 랭킹: 주간/월간 토글, 지급완료 총액 순
- 메뉴: 기존 서브페이지 재사용
